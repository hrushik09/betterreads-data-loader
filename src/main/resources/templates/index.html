BetterReads Data Loader

This application will upload the authors and books data to the Apache Cassandra database hosted at DataStax.

Open Library provides monthly bulk data dumps for authors (around 3GB) and books (around 13GB). These .txt files are split into smaller files and uploaded on AWS S3.
The Test option below will establish connection with AWS, read test files for both authors and works and upload that data to the Cassandra database.
The Prod option below will establish connection with AWS, read prod files for both authors and works and upload that data to the Cassandra database.
